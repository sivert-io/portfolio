Holo Notes explores how musicians can sculpt sound in mid-air. I led the interaction design, front-end implementation, and motion direction.

### Prototype stack

- WebXR + Three.js for depth-aware visuals.
- Web Audio API node graph for modular routing.
- Zustand + React for state management and performance tuning.

![Detail view of the modular audio patching interface](https://images.unsplash.com/photo-1511379938547-c1f69419868d?auto=format&fit=crop&w=1400&q=80)

The final installation premiered at an audiovisual art festival, where attendees could capture their sessions. We open-sourced the core engine, and it now powers workshops for creative coding students.
