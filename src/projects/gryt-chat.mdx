Gryt Chat is an open-source, self-hosted communication platform for voice, text, and video. It was built around privacy and user ownership, with features like HD voice/video, screen sharing, multi-server support, and accessibility-focused UX.

This project was especially important for me because I designed the infrastructure and overall architecture myself. That gave me hands-on experience acting as both developer and architect: defining service boundaries, planning deployment, and designing a system that could support real-time communication reliably.

From a technical perspective, Gryt combined a React/TypeScript + Electron client, a Node.js signaling layer, and a Go-based SFU for WebRTC media. I also worked with authentication and trust design using Keycloak, OIDC/PKCE, and challenge-response authentication, plus storage and deployment concerns using SQLite, MinIO/S3, Docker Compose, and Kubernetes/Helm.

Gryt gave me experience in architecture, real-time systems, authentication design, and self-hosted infrastructure — not just building features, but designing how the whole platform should work.

**Tech:** React, TypeScript, Electron, Node.js, Socket.IO, Go, WebRTC, Keycloak, OIDC/PKCE, SQLite, MinIO/S3, Docker Compose, Helm, Kubernetes
