# WebSocket Voice

<ProjectMeta repo={meta.repo} />

## Overview

WebSocket Voice is a proof-of-concept real-time voice system.

It explores how voice communication systems work under the hood.

## Stack

- React
- TypeScript
- WebRTC
- WebSockets
- Go (Pion)

## Goals

- experiment with audio transport
- explore SFU architecture
- understand WebRTC signaling flows
