FBX Action Exporter is a Blender add-on I built to streamline animation export workflows for game development. Instead of manually exporting animations one by one, the tool batch-exports actions as separate FBX files, making it much easier to move animation libraries into engines like Roblox, Unity, and Unreal.

What made this project useful was that it solved a very practical pipeline problem. It was less about flashy features and more about reducing repetitive work, improving consistency, and making asset export faster for real production workflows. That kind of tooling work is something I enjoy a lot: finding friction in a workflow and turning it into something reliable and reusable.

The add-on integrates directly into Blender’s UI and exposes configurable export settings for transforms, geometry, and armatures, so it can adapt to different engine requirements. It gave me experience working with Blender’s Python API, editor tooling, and the kinds of small but high-impact tools that improve day-to-day developer and content workflows.

This project gave me hands-on experience with pipeline tooling, automation, and building practical utilities for game development workflows.

**Tech:** Python, Blender API, Blender add-on development, FBX export automation, game-development pipelines, animation tooling
