# Matchzy Auto Tournament

<ProjectMeta repo={meta.repo} />

![Bracket Screenshot](/projects/matchzy/bracket.png)

## What problem does it solve?

Running online tournaments requires coordinating:

- brackets
- game servers
- match state
- results

This project automates the entire pipeline.

## Why I built it

After hosting multiple community tournaments I realized the process was extremely manual.

This system automates:

- bracket progression
- server allocation
- match lifecycle

## Architecture

Stack:

- **TypeScript**
- **React**
- **Node.js**
- **CS2 MatchZy plugin**

Flow:

```

Tournament Engine
↓
Match orchestration
↓
Server allocation
↓
CS2 match servers

```

## Features

- automatic bracket progression
- match server allocation
- live match tracking
- API for tournament tools
