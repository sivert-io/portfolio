Handy Unity Scripts is a small collection of Unity tools and utility scripts I built to remove friction from day-to-day development work. Rather than focusing on one large system, the project was about solving practical workflow problems with reusable editor and runtime helpers.

The repository includes tools like a custom editor window for bulk-adding and managing Dynamic Bone components across multiple objects, as well as runtime audio utilities for spawning randomized sound sources inside configurable 3D spaces. These were the kinds of tools that made repetitive tasks faster, more consistent, and easier to manage inside real Unity projects.

What I like about this project is that it reflects a pattern that has followed me through a lot of my work: identifying friction in a workflow and building small, focused tools to remove it. Even though the scripts are simple compared to larger systems, they show the same engineering mindset — practical automation, better usability, and reusable internal tooling.

This project gave me hands-on experience with Unity editor scripting, runtime systems, developer tooling, and building quality-of-life improvements for game development workflows.

**Tech:** Unity, C#, Unity EditorWindow tooling, runtime utilities, audio systems, workflow automation, game-development tooling
