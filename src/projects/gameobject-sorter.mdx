GameObject Sorter is a Unity editor tool I built to make large scene hierarchies easier to organize. Instead of manually restructuring objects by hand, the tool helps sort and clean up GameObjects directly inside the editor, making scene management faster and less repetitive.

What made this project useful was that it focused on workflow rather than runtime features. It was built to solve a very practical development problem: when projects grow, scene hierarchies become messy quickly, and small tooling improvements can save a lot of time. This project reflects the kind of engineering I enjoy most — finding friction in day-to-day work and building something simple, reliable, and reusable to remove it. ([GitHub README](https://github.com/sivert-io/gameobject-sorter))

The project also gave me experience building custom Unity editor tooling in C#, integrating directly into the editor workflow, and designing utility software that improves developer productivity without adding complexity to the actual game code.

This project gave me hands-on experience with editor scripting, workflow automation, and building small tools that improve day-to-day game development.

**Tech:** Unity, C#, Unity editor scripting, workflow automation, GameObject hierarchy tooling
